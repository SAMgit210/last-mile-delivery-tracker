# Last Mile Delivery Tracker

A full-stack logistics and delivery management system.

## Features

- User Authentication (Register/Login)
- Zone Management
- Rate Card Management
- Order Management
- MongoDB Atlas Integration
- REST APIs with Express.js
- React Frontend

## Tech Stack

### Frontend
- React
- Vite
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs

## API Endpoints

### Authentication
POST /api/auth/register

POST /api/auth/login

### Zones
POST /api/zones

GET /api/zones

### Rate Cards
POST /api/ratecards

GET /api/ratecards

### Orders
POST /api/orders

GET /api/orders

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
